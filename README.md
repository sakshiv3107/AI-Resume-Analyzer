## рџ“– Overview

Upload a resume in PDF format, optionally paste in a job description, and get back a full ATS-style report вЂ” score, match percentage, missing keywords, detected skills, and prioritized suggestions for improvement, all generated by Google Gemini.

Registered users get a saved history of past analyses; guests can run an analysis without an account, though nothing is saved.

---

## вњЁ Features

- **Resume Upload** вЂ” drag-and-drop or file picker, PDF validation, and inline preview
- **AI Resume Analysis** вЂ” Gemini evaluates quality, ATS compatibility, formatting, skills, experience, projects, education, and certifications
- **ATS Score** вЂ” an overall score out of 100
- **JD Match Percentage** вЂ” compares the resume against a pasted job description and flags missing requirements
- **Keyword Analysis** вЂ” surfaces missing keywords, existing keywords, and relevant technologies
- **Skill Detection** вЂ” auto-extracts skills found in the resume
- **AI Suggestions** вЂ” prioritized (high / medium / low) improvement tips
- **Strengths & Weaknesses** breakdown
- **AI-generated Resume Summary**
- **Authentication** вЂ” email/password and Google login via Supabase, plus a guest mode
- **Analysis History** вЂ” view, search, and filter past reports (logged-in users)
- **Downloadable Report** вЂ” export the full analysis
- **Fully Responsive** вЂ” desktop, tablet, and mobile

## вљ™пёЏ Installation

**1. Clone the repository**
```bash
git clone https://github.com/sakshiv3107/AI-Resume-Analyzer.git
cd AI-Resume-Analyzer
```

**2. Install dependencies**
```bash
npm install
```

**3. Set up environment variables**

Create a `.env` file in the project root:
```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_key
VITE_GEMINI_API_KEY=your_gemini_api_key
```

**4. Run the dev server**
```bash
npm run dev
```
